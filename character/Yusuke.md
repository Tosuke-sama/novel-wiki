## 外貌特征

- **人形**：鸟首人身，身穿西装
- **鸟形**：白色小鸟
- **特征**：鹰嘴似的鼻子，深邃漆黑的眼睛
- **羽毛**：头部顶部延伸到后脑勺耷拉着几根稀疏的白色羽毛

---

## 能力详解

### 预知能力
> 能够短暂感知到时间的流向，类似蜘蛛侠的感应

| 特性 | 描述 |
|------|------|
| **原理** | 感知未来几秒内的危险 |
| **限制** | 只能感知，不能改变 |
| **应用** | 战斗回避、危机预警 |

### 时间跃迁
> 使用跃迁羽进行短时间时间回溯

| 特性 | 描述 |
|------|------|
| **材料** | 跃迁羽（时间鸟身上的羽毛） |
| **时长** | 几秒到十几秒 |
| **限制** | 跃迁羽离体后大部分会自动消散 |
| **消耗** | 每次跃迁消耗一根跃迁羽 |

## 人物经历

### 出生与灾难
> 未孵化时双亲在围猎行动中丧生

- **事件**：时间鸟被猎杀，双亲死亡
- **原因**：跃迁羽具有极高价值
- **结果**：尹介成为孤儿

### 被蕾妤所救
> 被蕾妤发现并带回圣鹿蒂亚学院

- **救命恩人**：蕾妤
- **成长地点**：圣鹿蒂亚学院
- **学习内容**：人类文明、语言、礼仪

### 加入岚队
> 成为圣鹿蒂亚执行部岚队队员

- **职位**：岚队队员
- **任务**：辅助蕾妤执行任务
- **搭档**：蕾妤

## 角色分析

### 性格特点
- **沉着冷静**：执行任务时保持冷静
- **善良**：心地善良，愿意帮助他人
- **自尊心强**：不喜欢被当作鸟类看待
- **忠诚**：对蕾妤和圣鹿蒂亚忠诚

### 成长弧线

2. **学生** → 学习人类文明
3. **队员** → 加入岚队执行任务
4. **伙伴** → 与叶瑾介等人建立信任
