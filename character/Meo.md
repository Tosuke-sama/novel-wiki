## 能力详解

### 记忆操控
> 能够操控自己和他人的记忆

| 特性 | 描述 |
|------|------|
| **范围** | 自身及周围人 |
| **效果** | 修改、删除、添加记忆 |
| **限制** | 需要物理接触或近距离 |
| **反噬** | 自身也会失去记忆 |

### 健忘症
> 能力反噬导致的副作用

| 症状 | 描述 |
|------|------|
| **短期记忆丧失** | 刚发生的事情很快忘记 |
| **依赖笔记** | 需要靠笔记记录重要事情 |
| **情感记忆** | 情感相关的记忆相对牢固 |

- **天真**：长期生活在内室，对外界不了解
