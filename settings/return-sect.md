## 组织背景

### 分裂原因
> 对圣鹿蒂亚与王国政府合作不满

- **导火索**：圣鹿蒂亚与王国政府临时合作
- **理念冲突**：认为违背了组织理念
- **分裂**：联合起来成立新组织
- **发展**：成员数量不断上涨

### 核心理念
> 世界线融合是世界自我修复

| 理念 | 描述 |
|------|------|
| **自然选择** | 世界线融合是世界的选择 |
| **反对干预** | 不应该阻止融合 |
| **回归统一** | 让世界回归统一状态 |
| **宿命论** | 融合是不可避免的宿命 |
