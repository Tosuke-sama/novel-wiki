# 世界线穿越原理

> 穿越世界线的科学原理

---

## 基本信息

| 项目 | 内容 |
|------|------|
| **名称** | 世界线穿越 |
| **类型** | 空间穿越、时间穿越 |
| **代表** | 蕾妤（空间）、尹介（时间） |
| **基础** | 量子力学、高维物理 |

---

## 核心概念

### 世界线
> 代表一种可能性的命运轨迹

| 特性 | 描述 |
|------|------|
| **数量** | 无数多个 |
| **产生** | 每次选择产生分歧 |
| **状态** | 稳定态、纠缠态、融合态 |
| **载体** | 电影、小说、照片、文字等 |

### 孪生世界线
> 处于量子纠缠状态的两条世界线

| 特性 | 描述 |
|------|------|
| **关系** | 像人类的双胞胎 |
| **概率** | 低得可怕，是奇迹 |
| **状态** | 不稳定的随机态 |
| **危机** | 融合后只留下一份 |

### 量子纠缠
> 孪生世界线的物理基础

| 特性 | 描述 |
|------|------|
| **原理** | 量子力学现象 |
| **表现** | 两个世界相互影响 |
| **观测** | 自己无法观测自己 |
| **融合** | 相遇即坍塌成稳定态 |

---

## 穿越类型

### 空间穿越
> 在不同世界线之间移动

| 项目 | 内容 |
|------|------|
| **代表** | 蕾妤 |
| **能力** | 世界线跃迁 |
| **原理** | 突破世界线限界 |
| **效果** | 与不同世界线的人沟通 |

### 时间穿越
> 在同一世界线内时间移动

| 项目 | 内容 |
|------|------|
| **代表** | 尹介 |
| **能力** | 时间跃迁 |
| **原理** | 使用跃迁羽 |
| **效果** | 回溯几秒到十几秒 |

### 时空穿越
> 同时穿越空间和时间

| 项目 | 内容 |
|------|------|
| **代表** | 凛希（可能） |
| **能力** | 时空投影 |
| **原理** | 释放投影到对应时间线 |
| **效果** | 寻找物品 |

---

## 穿越条件

### 能力要求
> 需要特殊能力

| 类型 | 描述 |
|------|------|
| **觉醒者** | 因世界线引力变异 |
| **世界线观测者** | 极其罕见 |
| **THE KEY** | 拥有特殊基因 |
| **异灵契约者** | 与异灵缔结契约 |

### 物质要求
> 需要特殊物质

| 物质 | 描述 |
|------|------|
| **跃迁羽** | 时间鸟的羽毛 |
| **引导物** | 寻找物品时需要 |
| **基因** | THE KEY 基因 |
| **设备** | 高科技设备辅助 |

---

## 穿越限制

### 能量限制
> 穿越需要消耗能量

| 限制 | 描述 |
|------|------|
| **体力消耗** | 穿越会消耗体力 |
| **次数限制** | 不能无限穿越 |
| **冷却时间** | 需要恢复时间 |
| **材料消耗** | 跃迁羽等会消耗 |

### 规则限制
> 世界线三大原则

| 原则 | 描述 |
|------|------|
| **不可观测** | 自己无法观测自己 |
| **随机态** | 未被观测时处于随机状态 |
| **融合** | 相遇即融合，不可逆转 |

---

## 穿越风险

### 身份暴露
> 可能被当地人发现

| 风险 | 描述 |
|------|------|
| **外貌差异** | 可能被当作异类 |
| **行为差异** | 行为习惯可能暴露 |
| **知识差异** | 拥有未来知识 |
| **能力暴露** | 特殊能力被发现 |

### 世界线干扰
> 可能影响世界线稳定

| 风险 | 描述 |
|------|------|
| **蝴蝶效应** | 小改变可能引起大影响 |
| **因果悖论** | 可能产生因果矛盾 |
| **融合加速** | 可能加速世界线融合 |
| **身份混乱** | 可能遇到另一个自己 |
