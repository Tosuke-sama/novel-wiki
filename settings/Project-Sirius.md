# 天狼星计划（Project Sirius）

> 克隆 THE KEY 的秘密计划

---

## 基本信息

| 项目 | 内容 |
|------|------|
| **名称** | 天狼星计划 |
| **时间** | 对抗时代 271 年启动 |

| **地点** | 王国政府科学院 |
| **目的** | 制造 THE KEY 的替代品 |

---

## 计划背景

### 世界线融合危机
> 因帕尼亚文明灭亡的教训

- **前代文明**：因帕尼亚
- **灭亡原因**：世界线融合
- **遗留**：阻止融合的方法被封存
- **开启条件**：需要 THE KEY 基因

### THE KEY 的重要性
> 唯一能开启"门"的基因

- **原生携带者**：名伊王子
- **风险**：让王子冒险太危险
- **解决方案**：制造克隆体替代

---

## 计划执行

### 克隆方法
> 采用克隆原生技术

| 步骤 | 描述 |
|------|------|
| **取样** | 名伊出生后第 5 分钟取样 |
| **培养** | 移植到培养容器 |
| **成长** | 加速成长到目标年龄 |
| **教育** | 艾斯德教授亲自教育 |

### 实验对象
> 里克·托斯克

| 项目 | 内容 |
|------|------|
| **姓名** | 里克·托斯克 |
| **身份** | 名伊王子的克隆体 |
| **年龄** | 15 岁（实验室生活时间） |
| **基因** | 与名伊相同的 THE KEY 基因 |

### 实验室生活
> 托斯克的 15 年

- **教育者**：艾斯德教授
- **生活范围**：实验室及周边
- **心理状态**：渴望外面的世界
- **计划**：经常策划逃出实验室

---

## 计划结果

### 出逃事件
> 对抗时代 286 年

- **契机**：蕾妤小队侵入王国政府
- **帮助者**：艾斯德教授（故意放走）
- **结果**：托斯克成功出逃
- **后果**：艾斯德教授被囚禁

### 教授被囚
> 艾斯德教授的结局

- **罪名**：放走实验对象
- **判决**：终身监禁
- **后续**：被圣鹿蒂亚解救
- **现状**：未知
